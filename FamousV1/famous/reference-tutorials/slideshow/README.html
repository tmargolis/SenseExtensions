Famo.us Slideshow Tutorial assets

Tutorial

The tutorial accompanying these assets is available at http://famo.us/university/famous-101/slideshow/.

Start

The start folder contains the materials you will build off of for the tutorial. Opening index.html should yield a blank page. The tutorial will teach you how to edit and create code in the src directory.

Final

The final folder shows the state of the code after you have finished the tutorial. It can serve as an answer key when needed.
