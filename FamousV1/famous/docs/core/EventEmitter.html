EventEmitter

EventEmitter represents a channel for events.

Overview

Methods

emit(type, event)

Trigger an event, sending to all downstream handlers listening for provided 'type' key.

Parameters
type
String
event type key (for example, 'click')
event
Object
event data
Returns

callback
